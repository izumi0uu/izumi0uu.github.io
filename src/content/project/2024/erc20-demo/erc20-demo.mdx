## 项目概述

ERC20 Demo 是一个展示以太坊ERC20代币标准实现的完整项目。项目包含智能合约开发、部署以及前端Web3集成，为学习区块链开发提供了一个实用的示例。

## 技术栈

- **智能合约**: Solidity
- **区块链平台**: Ethereum
- **前端框架**: React + TypeScript
- **Web3集成**: Ethers.js / Web3.js
- **开发工具**: Hardhat / Truffle

## 主要功能

1. **ERC20代币合约**

   - 标准ERC20接口实现
   - 代币铸造和销毁
   - 转账和授权功能

2. **Web3前端界面**

   - 钱包连接
   - 余额查询
   - 代币转账
   - 交易历史

3. **智能合约部署**
   - 测试网部署脚本
   - 合约验证和发布

## 学习价值

这个项目非常适合区块链开发初学者，可以学习到：

- Solidity智能合约开发
- ERC20标准的实现
- Web3前端集成
- 区块链交互原理
